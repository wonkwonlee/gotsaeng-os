# GotSaeng OS v0.10 Install Guide

## Requirements

- Node.js 20 or newer
- npm

## Install the CLI

```bash
npm install -g @gotsaeng/cli
```

## Verify Installation

```bash
gotsaeng doctor
```

Expected result:

```text
GotSaeng OS Doctor

CLI version: 0.10.0
Write permission: ok
Core scanner: ok
```

## Compile a Vault

```bash
gotsaeng compile ./notes --output ./context-pack --project "My Project" --stale-days 90
```

Generated output includes:

- `PROJECT_CONTEXT.md`
- `MEMORY_SNAPSHOT.md`
- `DECISION_LOG.md`
- `ACTION_BACKLOG.md`
- `RISK_REGISTER.md`
- `OPEN_QUESTIONS.md`
- `STALE_CONTEXT.md`
- `SOURCE_PROVENANCE.md`
- `CONFIDENCE.md`
- `CONTRADICTIONS.md`
- `MEMORY_DIFF.md`
- `CONTEXT_MANIFEST.json`
- `COMPILE_REPORT.json`

## Validate a Vault

```bash
gotsaeng validate ./notes
gotsaeng validate ./notes --strict
```

Compatibility mode reports warnings for common real-world Obsidian vault metadata. Strict mode enforces the canonical GotSaeng OS schema.
