# Workflows

## Weekly Review Context Pack

Use a weekly review vault to surface open actions, stale commitments, risks, and questions.
Compile it before planning the next week.

## Project Handoff to LLM

Compile a project vault and provide `PROJECT_CONTEXT.md`, `MEMORY_SNAPSHOT.md`, and
`DECISION_LOG.md` to ChatGPT, Claude, Gemini, or a local LLM as a source-aware handoff.

## Career Strategy Vault

Track goals, constraints, decisions, risks, assumptions, and open questions across job search,
portfolio, and long-term skill development notes.

## Technical Research Vault

Compile research notes into durable facts, assumptions, insights, questions, and risks before
making architecture decisions.

## ADHD-Aware Reflection Workflow

Use explicit markers to reduce the cost of recovering context after interruption. The workflow
is ADHD-aware, not ADHD-limited: it helps anyone who needs a clean recovery surface.

## Future Engineering Ops Memory Workflow

Future versions may compile team decisions, incident logs, project risks, and execution records
into source-aware engineering memory packs.
